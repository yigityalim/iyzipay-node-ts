**Iyzipay CLI**, Iyzico ödemeleriyle çalışırken Geliştirici Deneyiminizi (DX) iyileştirmek için tasarlanmıştır. Terminalinizden doğrudan bağlantıları test etmenize, API anahtarlarını yönetmenize ve bileşenler oluşturmanıza olanak tanır.

## Kurulum

CLI'ı global olarak kurmanıza gerek yoktur. Paket yöneticinizin çalıştırma komutunu (örn. `npx` veya `pnpx`) kullanarak doğrudan çalıştırabilirsiniz.

Ancak, projenize de ekleyebilirsiniz:

```package-install
npm install iyzipay-node-ts
```

## Hızlı Başlangıç

Kullanılabilir seçenekleri görmek için yardım komutunu çalıştırın:

```bash
npx iyzi --help
```
