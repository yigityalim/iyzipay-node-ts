Install the package via your preferred package manager.

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tab value="npm">
  ```bash
  npm install iyzipay-node-ts zod
  ```
  </Tab>
  <Tab value="pnpm">
  ```bash
  pnpm add iyzipay-node-ts zod
  ```
  </Tab>
  <Tab value="yarn">
  ```bash
  yarn add iyzipay-node-ts zod
  ```
  </Tab>
  <Tab value="bun">
  ```bash
  bun add iyzipay-node-ts zod
  ```
  </Tab>
</Tabs>

## Requirements

- Node.js 18+
- TypeScript 5.0+ (Recommended)
- Zod 3.0+ (Peer Dependency)
