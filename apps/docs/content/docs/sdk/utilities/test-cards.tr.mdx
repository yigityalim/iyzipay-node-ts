Sandbox testleri için önceden tanımlanmış kartlara doğrudan SDK üzerinden erişin.

## Kullanım

```typescript
import { TestCards } from 'iyzipay-node-ts';

// Ödeme isteklerinizde doğrudan kullanın
const payment = await iyzipay.payment.create({
  // ...
  paymentCard: TestCards.Success.GarantiDebit
});
```
