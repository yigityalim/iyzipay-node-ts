Access predefined cards for sandbox testing directly from the SDK.

## Usage

```typescript
import { TestCards } from 'iyzipay-node-ts';

// Use directly in your payment requests
const payment = await iyzipay.payment.create({
  // ...
  paymentCard: TestCards.Success.GarantiDebit
});
```
