# Introducing iyzipay-node-ts

We are thrilled to announce the release of **iyzipay-node-ts**, a reimagined Iyzico client built for the modern JavaScript ecosystem.

If you've ever struggled with the official library's lack of TypeScript support, callback hell, or inability to run on edge environments like Cloudflare Workers, this package is for you.

## Why Re-invent the Wheel?

The official `iyzipay` library was built in a different era of Node.js. Today, we need robust typing, `async/await`, and edge compatibility.

## Killer Features

### 1. Zero-Config Validation

We use **Zod** under the hood. You don't need to guess which fields are required or what the card number format should be. We export these validators for you to use in your frontend too!

```tsx tab="React Hook Form Example"
import { useForm } from 'react-hook-form';
import { zodResolver } from '@hookform/resolvers/zod';
import { z } from 'zod';
import { IyzipayValidators } from 'iyzipay-node-ts';

const formSchema = z.object({
  // Built-in TCKN & Luhn Algorithm checks
  identityNumber: IyzipayValidators.tcKimlikNo, // [!code highlight]
  cardNumber: IyzipayValidators.creditCard,     // [!code highlight]
});

```

### 3. Modern Async API

Say goodbye to callbacks. Everything is Promise-based and fully typed.

```typescript title="payment.ts"
import { iyzipay, Currency, PaymentGroup } from 'iyzipay-node-ts';

const client = iyzipay({ ...config });

// Clean, readable, and typed
const { data, error } = await client.payment.create({
  price: '120.00',
  currency: Currency.TRY,
  paymentGroup: PaymentGroup.PRODUCT,
  // ...other parameters
});

if (error) {
  console.error(error.errorMessage); // [!code error]
  return;
}

console.log(data.paymentId); // [!code success]

```

## Getting Started

Ready to upgrade your payment infrastructure? It takes less than 30 seconds.

<Steps>
<Step>
### Install the package

```npm
npm install iyzipay-node-ts
```

</Step>

<Step>
### Initialize

Create your client instance. We recommend using environment variables.

```typescript
import { createClient } from 'iyzipay-node-ts';

export const client = createClient({
  apiKey: process.env.IYZIPAY_API_KEY,
  secretKey: process.env.IYZIPAY_SECRET_KEY,
  baseUrl: 'https://sandbox-api.iyzipay.com'
});
```

</Step>
